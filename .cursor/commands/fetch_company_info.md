# Fetch Company Info

## Overview
회사 폴더 내의 대화 내역(quotes, meeting_scripts)을 분석하여 찾으려는 회사의 비즈니스 프로필을 요약합니다. 이때 외주 개발과 관련된 맥락(리트머스와의 협업 등)은 배제하고, 해당 회사가 어떤 사업을 하는지에만 집중하여 정보를 추론합니다. 추론된 키워드를 바탕으로 웹 검색을 통해 대상 회사를 확정하고, Perplexity API를 사용하여 매출, 연혁, 주요 사업 등 구체적인 정보를 수집하여 정리합니다.

## Steps

### 1. 분석 및 프로필 추론 (Inference)
- 클라이언트 폴더(`YY_MM_CLIENT_NAME/`) 내의 주요 파일(`quotes/`, `meeting_scripts/`)을 읽습니다.
- **외주 개발 맥락 제거**: "개발 요청", "견적", "기술 스택" 등 리트머스와의 협업 내용은 무시합니다.
- **순수 비즈니스 분석**: 
  - 이 회사는 무엇을 팔거나 제공하는가?
  - 누구를 대상으로 하는가? (B2B, B2C, 특정 업종 등)
  - 시장에서의 위치나 해결하려는 문제(Pain Point)는 무엇인가?
- 분석 결과를 바탕으로 **[찾으려는 회사의 가상 프로필]**을 작성합니다. (이름, 업종, 핵심 서비스 키워드 포함)

### 2. 대상 회사 확정 검색 (Identification)
- 추론한 프로필과 키워드를 바탕으로 웹 검색을 진행합니다.
- **검색 전략**:
  - 폴더명 또는 대화 중 언급된 실제 회사명 검색
  - 업종 + 핵심 서비스 + 특정 지역 등 조합 검색
  - 결과 중 대화 내용과 가장 일치하는 서비스나 회사 사이트를 찾습니다.
- 만약 정확한 회사를 찾기 어렵다면, 대화 내용에서 묘사된 것과 가장 유사한 기존 서비스나 경쟁사를 후보로 선정합니다.
- 정리한 내용을 비판적으로 사고해, 불분명한 정보나 가정은 제거합니다.

### 3. Perplexity 기반 구체적 정보 수집 (Deep Dive)
- 확정된(또는 가장 가능성 높은) 회사명을 대상으로 Perplexity API 검색을 수행합니다.
- **실행 명령**:
  - `python3 .cursor/scripts/perplexity_search.py "[회사명] 회사 개요 매출 연혁 직원 수 주소"`
  - `python3 .cursor/scripts/perplexity_search.py "[회사명] 주요 사업 분야 및 서비스 상세 정보"`
- 검색 결과에서 다음 항목을 추출합니다:
  - 기본 정보: 정확한 법인명, 대표자, 설립일, 본사 주소
  - 규모: 매출액(최근), 직원 수, 투자 단계(해당 시)
  - 사업: 주요 서비스 라인업, 핵심 기술, 주요 고객사
  - 기타: 공식 웹사이트 URL, 최근 뉴스 등

### 4. `company_info.md` 생성
- 수집된 정보를 바탕으로 파일 생성 (웹 검색 결과만 사용).
- 정보가 없는 항목은 "정보 없음"으로 표시하되, 대화 내역의 추측성 정보로 채우지 않습니다.
- 정리한 내용을 비판적으로 사고해, 불분명한 정보나 가정은 제거합니다.

## Output Format

```markdown
# [회사명] 회사 정보

_Last updated: YYYY-MM-DD_

## 회사 개요
- 회사명: [공식 법인명 또는 서비스명]
- 업종/사업 분야: [업종]
- 대표자: [대표자명]
- 설립일: [YYYY-MM-DD]

## 비즈니스 프로필 (추론 기반)
- 주요 서비스: [서비스 명칭 및 핵심 기능]
- 타겟 고객: [주요 고객층]
- 비즈니스 모델: [수익 구조 또는 서비스 형태]

## 매출 및 규모
- 최근 매출: [매출 정보]
- 직원 수: [인원 정보]
- 본사 위치: [상세 주소]

## 회사 연혁
- [YYYY.MM] [주요 사건]
- [YYYY.MM] [주요 사건]

## 주요 사업 및 서비스
- **[서비스/사업명 1]**: [상세 설명]
- **[서비스/사업명 2]**: [상세 설명]

## 정보 출처
- 웹사이트: [URL]
- 검색 전략 및 키워드:
  - 프로필 추론 키워드: [키워드 1, 키워드 2...]
  - Perplexity 쿼리: [사용된 쿼리]
- 참고: [유사 회사 정보를 사용한 경우 해당 사유 명시]
```

## Checklist
- [ ] 외주 개발 맥락을 제외한 순수 비즈니스 프로필을 작성했는가?
- [ ] 추론된 키워드로 대상 회사를 특정하기 위한 검색을 수행했는가?
- [ ] 대화 내역과 가장 부합하는 실제 회사/서비스를 선택했는가?
- [ ] Perplexity API를 통해 구체적인 수치(매출, 인원 등)를 조회했는가?
- [ ] 모든 정보는 웹 검색 결과를 바탕으로 작성되었는가? (대화 내역은 키워드 추출용으로만 사용)
- [ ] `company_info.md`를 지정된 포맷에 맞춰 생성했는가?
