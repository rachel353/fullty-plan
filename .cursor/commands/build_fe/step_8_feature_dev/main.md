# Step 8: 기능 단위 순차 개발 (Loop)

## Overview
Step 8은 `docs/dev_plan.md`의 순서대로 기능(F01, F02, …)을 하나씩 개발하는 반복 루프입니다. 각 기능마다 **명세(8-1) → UI/UX(8-2) → Schema/Mock(8-3) → UI 구현(8-4) → 검증(8-5)** 순서로 진행합니다.

## Steps
1. `docs/dev_plan.md`에서 “다음 기능”을 하나 선택합니다.
2. 선택한 기능에 대해 아래 순서로 진행합니다.
   - 8-1: 개발 명세 구체화
   - 8-2: UI/UX 계획 구체화
   - 8-3: Schema & Mock Data
   - 8-4: UI 개발
   - 8-5: 동작 확인
3. 기능이 완료되면 `docs/dev_plan.md` 또는 별도 진행표에 상태를 업데이트하고, 다음 기능으로 이동합니다.
4. 모든 기능이 완료되면 Step 9(최종 검증)로 이동합니다.

## Checklist
- [ ] 현재 개발할 기능이 `docs/dev_plan.md`의 우선순위/의존성과 일치하는가?
- [ ] Step 8-1~8-5를 모두 수행했는가?
- [ ] 기능 검증 결과(8-5)가 남아 있는가?

## Output (진행표 예시)
필요 시, 아래 형태로 진행 상황을 문서/노트에 남깁니다.

