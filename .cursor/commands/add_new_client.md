# Add New Client

## Overview
새로운 클라이언트 폴더 구조를 생성합니다. 현재 연도와 월을 자동으로 사용하여 `YY_MM_CLIENT_NAME/` 형식의 폴더를 만듭니다.

## Steps
1. 클라이언트 이름 `[CLIENT_NAME]`과 현재 연도/월을 확인
2. `YY_MM_CLIENT_NAME/` 형식의 폴더 생성
3. 하위 폴더 생성: `quotes/`, `meeting_scripts/`, `appendix/`
4. `git status` 확인 → `git add` → `git commit`

## Checklist
- [ ] 폴더 구조가 올바르게 생성되었는가?
- [ ] `quotes/`, `meeting_scripts/`, `appendix/` 폴더가 모두 생성되었는가?
- [ ] 커밋까지 완료했는가?
