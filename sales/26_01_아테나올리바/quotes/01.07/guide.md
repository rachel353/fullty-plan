# Quote Generation Guide - 아테나올리바 - 01.07

## 1) 고객사 핵심 가치 및 우선순위

1. **비비노 수준의 스캔 경험**: 올리브오일 라벨을 촬영하면 즉시 제품 정보가 표시되는 심리스한 UX가 최우선
2. **OCR 응답 속도**: 기존 9-10초 → 3초 이내 응답이 필수 (계약서 명시 요구)
3. **기존 시스템 활용**: 카페24 기반 시스템과 연동 가능 여부 우선 확인 (연동 실패 시 신규 개발)
4. **단계적 접근**: POC로 기술 검증 후 본 개발 진행하는 리스크 최소화 전략 선호
5. **웹앱 형태 유지**: 기존 앱스토어 계정 활용하여 업데이트 배포 (네이티브 앱 별도 개발 불필요)

## 2) 중요 합의사항 요약

### POC 단계 (1차)
- OCR 라벨 스캔 성능 테스트 (정확도 및 응답 시간 검증)
- 카페24 기존 시스템과 연동 테스트
- 기간: 약 2주
- 비용: 별도 제안서에서 확정 (참고: 유사 건 약 300~500만원)

### 본 개발 (POC 성공 후)
- 올리브오일 라벨 스캔 기능 (비비노 유사)
- 커머스 플랫폼 (상품 조회, 장바구니, 결제)
- 리뷰 시스템
- 관리자 페이지 (상품 DB 관리, OCR 데이터 관리)
- 예상 비용: 핵심 플로우 (OCR + 구매) 5,000~6,000만원
- 전체 개발 기간: 약 6~7개월 (기획 포함)
- 배포 목표: 2026년 7~8월

### 유지보수 체계
- 1개월 무상 유지보수
- 이후 유상 (연간 20% 또는 시간제 선택)
- 주말/야간 긴급 대응 별도 협의

## 5) 기술적/기획적 제약사항 및 고려사항

### 기술적 제약
1. **OCR 응답 속도 3초 이내**: 계약서에 보장 조건 명시 필요
2. **RAG 기반 검색 우선**: AI 호출 최소화로 토큰 비용 절감
3. **라벨 DB 규모**: 현재 100~1,000개, 향후 최대 만 개 이내
4. **카페24 연동 불확실성**: 연동 실패 시 신규 코드 개발 전환

### 기획적 고려사항
1. **비비노 벤치마킹**: 스캔 → 결과 → 구매 연결 플로우 참고
2. **전문가 리뷰 지원**: 일반 사용자 리뷰 외 전문가 리뷰 구분
3. **빈티지별 라벨 관리**: 동일 제품도 빈티지에 따라 라벨 데이터 별도 관리

### 리스크 요소
- OCR 정확도 목표 수치 미확정 (POC에서 확정)
- AI 토큰 비용 구체적 산정 필요 (스캔 횟수별)
- 주말/야간 긴급 대응 계약 조건 미협의

## 6) 견적 적정성 평가 및 범위 조정 제안

### POC 단계
- **협의된 견적**: 300~500만원
- **현실적인 견적**: 약 325만원 (0.5MM × ₩6,500,000)
- **적정성 판단**: ✅ 적정
- **제안 사항**: 범위 내에서 충분히 진행 가능

### 본 개발 (핵심 플로우)
- **협의된 견적**: 5,000~6,000만원
- **현실적인 견적**: 약 5,200~5,850만원 (0.8~0.9MM × 10개월 = 8~9MM)

#### 요구사항별 맨먼스 추정
| 영역 | 요구사항 | 추정 MM |
|------|---------|---------|

- **현실적 견적**: 8.0 MM × ₩6,500,000 = **52,000,000원**
- **적정성 판단**: ✅ 적정 (협의 범위 5,000~6,000만원 내)

### 제안 사항
1. POC 300~400만원으로 제안 (안전 마진 확보)
2. 본 개발 5,200만원 기준으로 제안서 작성
3. 리뷰 시스템(REQ-106)은 선택적 기능으로 분리 가능 (비용 절감 옵션)
