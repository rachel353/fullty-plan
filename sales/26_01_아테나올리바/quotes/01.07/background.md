# 프로젝트 배경 - 아테나올리바

## 0) 개편안 정리 핵심

> 본 프로젝트는 첫 미팅 기반으로, 기존 협의 대비 개편안은 해당 없습니다.

**단계적 접근 전략 채택:**
- POC 단계에서 OCR 성능 및 카페24 연동 가능성 검증
- 본 개발은 POC 성공 후 진행
- 페이즈별 결과 확인 후 다음 단계 진행

## 1) 프로젝트 개요

**아테나올리바 올리브오일 스캔 서비스**는 사용자가 올리브오일 라벨을 카메라로 촬영하면 제품 정보를 즉시 확인하고 바로 구매까지 연결되는 커머스 플랫폼입니다.

비비노(Vivino) 와인 스캔 앱과 유사한 사용자 경험을 올리브오일 시장에 적용하여, 소비자의 구매 결정을 돕고 아테나올리바의 온라인 판매를 촉진합니다.

**핵심 가치:**
- 3초 이내 라벨 인식으로 심리스한 스캔 경험 제공
- 스캔 → 제품 정보 → 구매로 이어지는 원스톱 쇼핑 플로우
- 전문가 리뷰와 사용자 리뷰를 통한 신뢰성 있는 제품 정보 제공

## 2) 서비스 배경 및 필요성

### 현재 문제점 (Pain Point)
1. **올리브오일 정보 접근성 부족**: 소비자가 매장에서 올리브오일을 선택할 때 참고할 수 있는 정보가 제한적
2. **기존 OCR 시스템 성능 한계**: 현재 9-10초 소요되는 응답 시간으로 사용자 경험 저하
3. **구매 전환율 저조**: 제품 정보 확인과 구매가 분리되어 있어 구매 전환 기회 손실

### 시장 기회
- 비비노의 성공 사례: 와인 시장에서 스캔 기반 서비스로 큰 성공
- 올리브오일 시장에는 유사 서비스 부재 → 선점 기회
- 프리미엄 올리브오일 시장 성장세

### 도입 시 기대 효과
- 매장 내 즉시 제품 정보 확인 → 구매 결정 시간 단축
- 온라인 커머스 연동으로 구매 전환율 향상
- 라벨 DB 구축을 통한 올리브오일 정보 플랫폼으로 발전 가능

## 3) 프로젝트 주요 목표

- **비비노 수준의 UX**: 자연스럽고 직관적인 스캔 경험 구현
- **신뢰할 수 있는 정보**: 전문가 리뷰와 사용자 리뷰 통합
- **심리스한 구매 경험**: 스캔부터 결제까지 끊김 없는 플로우

## 5) 유의할 기술적/기획적 포인트

### 기술 스택 제약
- **웹앱 형태**: 기존 앱스토어 계정 활용 (PWA 또는 웹뷰 앱)
- **카페24 연동 우선**: 연동 실패 시 신규 커머스 백엔드 개발

### 외부 시스템 연동
| 시스템 | 용도 | 리스크 |
|--------|------|--------|
| 카페24 | 기존 커머스 시스템 | 연동 불가 시 신규 개발 필요 |
| PG사 | 결제 연동 | 결제 오류 시 긴급 대응 필요 |
| OCR/AI | 라벨 인식 | 토큰 비용 최적화 필요 |

### 성능 요구사항
- **OCR 응답 시간**: 3초 이내 (계약서 명시)
- **RAG 우선 검색**: AI 호출 최소화로 비용 절감
- **24시간 가용성**: 서버 다운 시 긴급 대응 체계 필요

### 보안/규정 준수
- 회원 정보 보호 (개인정보보호법)
- 결제 정보 보안 (PG사 보안 기준 준수)

### 운영 고려사항
- **유지보수 계약**: 1개월 무상 → 이후 유상 (연간 20% 또는 시간제)
- **긴급 대응**: 주말/야간 대응 별도 협의 필요
- **라벨 DB 확장**: 초기 100~1,000개 → 향후 최대 만 개
