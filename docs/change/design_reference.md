# Navigation Design Characteristics

## 1. Layout Structure
- Full-width desktop navigation bar optimized for **1920px canvas**
- Horizontal flex layout
- Content distributed using **space-between**
- Vertical alignment centered
- Designed as a **logo / main menu / utility actions** structure
- Stable global navigation suitable for premium brand websites
- Slim but breathable header height: **112px**

## 2. Spacing System
- Large top padding: **40px**
- Bottom padding: **20px**
- Horizontal inner spacing between groups: **60px**
- Strong emphasis on **airy spacing and premium whitespace**
- Avoid dense menu composition
- Maintain generous breathing room between navigation elements

## 3. Visual Mood
- Background color: **Soft sage green (#BADCB1)**
- Natural, muted, wellness-oriented palette
- Creates a calm, premium, organic brand feeling
- Suitable for:
  - clean beauty
  - wellness
  - skincare
  - lifestyle commerce
  - premium editorial ecommerce

## 4. UX Direction
- Prioritize **brand mood first, navigation second**
- Menu should feel elegant rather than information-heavy
- Use clear visual separation between left / center / right groups
- Maintain a balanced and calm horizontal rhythm
- Suitable for sticky desktop navigation behavior

## 5. Component System
- Auto-layout based component structure
- Flexible for dynamic menu item expansion
- Logo, menu links, and utility buttons should be independent reusable components
- Should scale naturally when menu items are added or removed
- Preserve spacing tokens consistently across all child components
