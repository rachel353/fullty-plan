# Change Analysis: Reader 하단 네비게이션 구조 변경

> 생성일시: 2026-01-07 12:00
> 상태: in-review

## 원문 (Raw Feedback)

Reader Pages 하단 네비게이션 변경
- "충전" 탭을 하단 네비게이션에서 삭제
- "충전" 기능을 "마이페이지" 탭 내부로 이동

Context: 현재 Reader Pages의 하단 네비게이션에는 "충전" 탭이 별도로 존재하며, /shop 라우트로 Ink 충전 페이지에 접근. 이를 마이페이지 내부 기능으로 통합하려 함.
