# A. 회원 / 권한 플로우

```
회원가입
→ 역할 선택
   - 일반 회원
   - 셀러 (개인 / 사업자)

일반 회원 가입
→ 이메일 가입 / 소셜 로그인
→ 이용 가능 기능
   - 상품 구매 / 렌탈
   - GET 요청 (구해주세요)
   - SELL 요청 (위탁 / 매입 신청) ← 직접 등록 아님, 풀티에 신청하는 것

셀러 가입
→ 이메일 가입 / 소셜 로그인
→ 셀러 유형 선택
   - 개인 셀러
   - 사업자 셀러
→ 필수 추가정보 입력 (소셜 로그인에도 해당)
→ 관리자 승인 심사
   - 심사 중 / 승인 완료 / 반려
→ 승인 완료 후 셀러 기능 활성화
   - 상품 등록 (풀티 사전 검수 후 노출)
   - 가격 입력 / GET (구해주세요) 응답
   - 렌탈 공급 설정
   - 정산 관리

예외
→ 셀러는 소셜 로그인 시에도 추가 필수정보 입력 필요
→ 모든 상품은 풀티 사전 검수 완료 후 노출
```

# C. 렌탈 플로우

```
상품 상세
→ 렌탈 기간 선택
→ 배송지 입력
→ 결제
→ 결제 완료
→ 배송 준비
→ 배송 완료
→ 렌탈 이용 중
   ※ 종료일 D-N일 전 알림 발송

렌탈 종료 시 분기
1) 연장 (누적 포함 최장 90일 이내)
2) 구매 전환 → 차액 결제 → 소유권 이전 → 가구 자산화 등록 가능
3) 회수
   → 직접 배송 / 풀티 픽업 선택
   → 회수 비용 결제
   → 회수 진행
   → 풀티 검수 및 등급 결정
   → 재판매 상품 자동 전환

렌탈료 계산 정책
- 최대 렌탈기간: 90일 (연장 포함)
- 7일까지: 판매가의 10% 고정
- 8일 이상: 구간별 일요율 적용
  · 8~14일: 판매가 × 0.0075 × 선택일수
  · 15~30일: 판매가 × 0.00609 × 선택일수
  · 31~60일: 판매가 × 0.0046 × 선택일수
  · 61~90일: 판매가 × 0.00307 × 선택일수
- 최종 금액 100원 단위 올림
- 연장 시 렌탈료 계산 기준 → 미정 (확인 필요)
- 구매 전환 차액 계산 기준 → 미정 (확인 필요)
```

# D. GET / SELL 플로우

```
GET과 SELL 모두 로그인 유저에게만 가능

GET
→ 유저 GET 요청 폼 작성
→ 마이페이지 저장 (유효기간 없음)
→ 관리자 GET 관리
→ 셀러 제안 ("해당 상품 있음" + 상품/가격 제시)
→ 유저 확인
→ 거래 (구매 플로우로 연결)

SELL
→ 유저 매입 / 위탁 신청
→ 크기 / 지역 입력 → 기본 픽업 배송비 자동 산출
→ 추가 작업 가능성 체크 (조립·설치 등 현장 확인 후 별도 청구 안내)
→ 기본 픽업 배송비 결제 (확정)
→ 일정 조율
→ 픽업
→ (추가 작업 발생 시) ~~현장 수납 + 관리자 어드민 금액 기록~~ 관리자 추가 결제 요청 발송 
→ 유저 확인 후 결제
	 - 미결제 시 2일 후 재알림
   - 3일 이상 미결제 시 → 상품 반송 처리 (반송 비용 부담 주체 확인 필요)
→ 검수
   - 반려 시 → 유저 통보 후 반송 처리
   - 통과 시 →
→ 최종 금액 제안 (매입가 / 위탁 조건)
→ 유저 수락 / 거절
   - 거절 시 → 반송 처리
→ 계약서 작성
→ 매입 / 위탁 분기

	→ 매입
		→ 풀티 소유권 이전
		→ 정산 완료
		→ 상품 등록 및 판매

	→ 위탁
		→ 풀티 입점 처리
		→ 판매 완료 시 수수료 차감 후 정산
```

# E. 유저 마이페이지 플로우 (상기 내용 반영)

```jsx
**주문 조회**
→ 배송 전
   - 배송 조회
→ 배송 완료
   - 구매 확정 버튼 (D-7~D-1 카운트다운 텍스트 표시)
   - 배송 완료 7일 후 자동 구매 확정
   - 반품 요청 (배송 완료 후 3일 이내)
     → 반품 접수
     → 관리자 반품 승인
     → 풀티 픽업 수거
     → 수거 완료
     → 반품 완료
   - 후기 작성 (리빙 라운지 페이지로 이동)
   - 배송 조회
→ 각 상품 주문 상세 보기
   - 주문 상품 내역
   - 결제 정보
   - 환불 정보

→ 렌탈중인 상품
	- 상품 내역 (상품 이미지, 브랜드, 상품명, 렌탈 시작-종료 날짜, 렌탈가)
	- 상품 누르면 상세 보기
	  - 최초 렌탈 기간
	  - 연장 이력 (연장 1차 / 2차 등)
	  - 현재 상태
	- 종료 7일 전: 다음 단계 선택 버튼 활성화
	   1) 연장 (드롭다운 날짜 선택 - 확인 필요)
	      - 연장 완료 시 연장 뱃지 표시
	      - 연장 이후에는 구매 전환 또는 회수만 가능
	   2) 구매 전환
	      - 판매가 - 렌탈료 납부 금액 = 차액 즉시 결제
	      - 소유권 이전
	      - 가구 자산화 등록 가능
	   3) 회수
	      - 풀티 픽업 수거
			  - 수거 현황 표시

→ 취소 / 반품 내역
	- 취소 내역 보기
	- 반품 내역 보기
		- 반품 상세 보기
		- 주문 상세 보기 (주문 조회의 주문 상세 내역으로 연결)

**GET 내역**
→ 신청한 상품 내역
→ 상품 누르면 신청 폼 내역 확인
**→ 셀러 제안 알림
   - 제안 확인 (상품 / 가격)
   - 수락 / 거절
   - 수락 시 구매 플로우로 연결**

**SELL 내역**
→ 상품 이미지, 브랜드, 모델명, 옵션, 제품 상태(등급)
→ 연결된 GET 상품 (없으면 - 표시)
→ 진행 상황 status
   접수 완료 → 배송비 결제 완료 → 픽업 대기 → 픽업 완료
   → 검수 중 → 반려 / 최종 금액 제안 확인 대기
   → 계약 완료
→ 매입 / 위탁 계약서
→ 관리 (상세보기 버튼 → 신청 폼 내역 확인)

**COLLECTION**
→ 위시 리스트
	- 찜한 상품 내역 (상품 사진 UI)
	   - 품절 상품은 SOLD OUT 뱃지 표시

→ 자산 리포트
	- 풀티에서 소유한 가구 전체 표시
	   - 소유권 이전 시점에 가구별 고유 ID 자동 생성
	   - 중고 / 렌탈 / 판매 이력 추적 가능
	- 등급별 현재 예상 가치 표시 (SS / S / A+ / A / B)
	   - 수식 기준 확인 필요
	- 보유 가구 합산 가치 실시간 차트 시각화
	- 특정 기간 경과 시 "지금 판매/렌탈 시 수익 OO원" 알림 발송
	
**리빙 라운지**
→ 리뷰 내역
   - 내가 작성한 리뷰 목록 (리뷰 누르면 수정 / 삭제)
→ 정보글 작성
   - 작성한 정보글 목록 (정보글 누르면 수정 / 삭제)

**계정설정**
→ 회원정보 수정
	- 계정 정보
	  - 아이디
	  - 비밀번호 (변경 버튼)
	- 배송지 목록
	- 회원탈퇴

**→ 알림 설정
	- 배송 상황 알림
	- 렌탈 종료 알림
	- GET 제안 알림
	- SELL 진행 상황 알림**

→ 나의 등급 / 쿠폰 / 풀티머니
	- 현재 등급 표시 (확인 필요 - 등급 기준 및 혜택)
	- 보유 쿠폰 목록 (확인 필요 - 쿠폰 발급 기준)
	- 풀티머니 보유 금액 표시
	- 적립 / 사용 이력
```

# F. 셀러 어드민페이지 플로우

```jsx
대시보드
→ 판매 중인 상품 수
→ 렌탈 중인 상품 수
→ 이번 달 예상 정산액
→ 전체 현황 요약 노출

상품 등록
→ 상품 등록 폼 작성
   - 브랜드 / 모델명 / 옵션 / 상태 / 이미지
   - 모델 선택 시 신품 최저가 자동 노출 (가이드 프라이싱)
   - 풀티 등급 선택 시 권장 판매가 가이드 제시
   - 가격 설정
     - 공급가 (직접 입력)
     - 판매가 (직접 입력)
       - S등급 입력가가 신품 최저가보다 높을 경우 등록 제한
     - 수수료 (풀티 어드민 설정값 n% 자동 입력 / 고정)
     - 배송비 (**풀티가 구매자에게 발송하는 배송비 - 확인 필요**)
     - VAT (직접 입력)
     - 실 정산액 자동 계산 및 표시
   - 렌탈 공급 여부 토글
     - 렌탈 ON 시 제공 가능한 최소 / 최대 일수 설정
     - 일수 입력하면 렌탈료 자동 산정
     - 예상 수익금 및 풀티 수수료 자동 표시
→ 등록 버튼 → 풀티 어드민에 신규 검수 요청 알림 발송
→ 풀티로 실물 배송 (**배송비 부담 주체 확인 필요**)
→ 풀티 검수
   - 반려 시 → 사유 안내 후 반송 / 수정 재등록 가능
   - 승인 시 → 풀티 플랫폼에 상품 노출
→ 구매 발생 시 풀티가 구매자에게 발송
→ 판매 중 상태 관리

GET 알림 설정
→ 관심 브랜드 / 카테고리 설정
→ 관련 GET 요청 알림 수신
→ 제안 발송

정산 관리
→ 정산 내역 조회
   - 판매 정산 / 렌탈 정산 구분 조회
   - 기간별 필터 (주간 / 월간 / 직접 설정)
→ 정산 상태
   - 정산 예정 / 정산 확정 / 지급 완료
→ 정산 상세
   - 상품명 / 판매가 / 수수료 n% / 실 정산액
   - 렌탈의 경우 렌탈 기간 및 렌탈료 표시
→ 정산 계좌 관리
   - 계좌 등록 / 수정
```
