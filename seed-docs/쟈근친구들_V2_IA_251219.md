# IA(Information Architecture)

```sql
**공통**
├─ 스플래시
├─ 인증
│  ├─ 로그인
│  ├─ 회원가입
│  └─ 약관 동의
├─ 프로필 설정
└─ 마이페이지
   ├─ 내 정보
   └─ 내 지갑
      ├─ 잔액 요약
      └─ 트랜잭션 내역
```

```markdown
**독자**
├─ 홈
│  ├─ 추천 배너
│  └─ 작품 리스트
│
├─ 탐색
│  ├─ 작품 목록
│  ├─ 검색 결과
│  └─ 필터/정렬
│
├─ 작품 상세
│  ├─ 작품 정보
│  └─ 에피소드 목록
│
├─ 에피소드 뷰어
│  ├─ 콘텐츠 뷰어
│  ├─ 액션 바
│  │  ├─ 제안하기
│  │  └─ 후원하기
│  └─ 크레딧
│     ├─ 메인 프로듀서
│     └─ 참여자 리스트
│
└─ 제안
   ├─ 제안 작성
   ├─ 제안 목록
   └─ 제안 상세
```

```markdown
**작가**
├─ 크리에이터 스튜디오
│  ├─ 에피소드 관리
│  │  ├─ 에피소드 목록
│  │  └─ 에피소드 편집
│  │
│  └─ 제안 관리
│     ├─ 제안 대시보드
│     └─ 제안 상세
│
├─ 정산
│  ├─ 예정 정산
│  └─ 출금 가능 금액
│
└─ 마이페이지
   ├─ 프로필
   └─ 지갑
```

```markdown
**관리자**
├─ 대시보드
│
├─ 결제 관리
│  ├─ Ink 충전 내역
│  └─ 결제 상세
│
├─ 정산 관리
│  ├─ 정산 요청 목록
│  └─ 정산 처리
│
├─ 회원 관리
│  ├─ 회원 목록
│  └─ 회원 상세
│
├─ 콘텐츠 관리
│  ├─ 작품 목록
│  └─ 에피소드 관리
│
└─ 운영 관리
   ├─ 신고/제재
   └─ 시스템 설정
```
